# Sales-Transaction-Analytics
A data analytics project focused on analyzing sales transactions to identify revenue trends, customer purchasing behavior, top-selling products, and key business performance metrics using Excel, SQL, and Power BI.
# Sales Transaction Analysis Dashboard

## Overview
This project analyzes sales transaction data to uncover trends, customer purchasing behavior, and key business insights. The analysis includes data cleaning, exploratory data analysis (EDA), and interactive dashboard creation to support data-driven decision-making.

## Objectives
- Analyze monthly sales performance
- Identify top-selling products
- Understand customer purchasing behavior
- Track key sales metrics
- Generate actionable business insights

## Tools Used
- Microsoft Excel
- SQL
- Power BI

## Key Tasks Performed
- Cleaned and prepared raw sales data
- Queried and transformed data using SQL
- Performed exploratory data analysis (EDA)
- Created KPIs and interactive dashboards
- Visualized sales trends and product performance
- Generated business recommendations based on findings

## Dashboard Highlights
- Total Sales
- Monthly Sales Trend
- Top-Selling Products
- Customer Segmentation
- Revenue by Category
- Interactive Filters

## Key Insights
- Identified peak sales periods.
- Determined the highest-performing products.
- Analyzed customer purchasing patterns.
- Provided recommendations to improve sales performance.
